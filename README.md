
## AskWise AI – Gemini Powered Conversational Assistant

A lightweight conversational AI web app built using **Streamlit** and **Google Gemini**, designed to provide fast, interactive question–answering and knowledge assistance in real time.

This project demonstrates how modern LLMs can be integrated into simple web interfaces to create powerful AI tools with minimal frontend overhead.

---

## 🚀 Features

* Conversational chat interface
* Powered by Google Gemini (LLM)
* Real-time streaming responses
* Lightweight Streamlit UI
* Secure API key handling via environment variables
* Quick setup and deployment
* Cloud deployable in minutes

## 📦 Installation

### 1. Clone repo

```bash
git clone <repo-url>
cd project-folder
```

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Add API Key

Create `.env` file or set environment variable:

```
GOOGLE_API_KEY=your_key_here
```

---

## ▶️ Run locally

```bash
streamlit run app.py
```

Open:

```
http://localhost:8501
```

---

## ☁️ Deploy

Recommended: Streamlit Community Cloud

Steps:

1. Push code to GitHub
2. Connect repo on Streamlit Cloud
3. Add GOOGLE_API_KEY in Secrets
4. Deploy

Live in 2 minutes.
